◆ ConvertToGreyscale()

GreyMatrix* ConvertToGreyscale ( DynamicMatrix *  dm,
int  channel 
)

Function that converts an RGB Matrix

  • dm to a Greyscale matrix and returns it.

Essentially, it iterates through the RGB data, converts it and stores it in a buffer. This buffer is then used as the new Greyscale Matrix's data.

  • channel is an argument that allows the user to select a channel of color into which the RGB Matrix will be split. If NONE is specified, the Greyscale image generated will be an overlap of the Red, Green and Blue Greyscale images.
